Query Parameters

  • recuperarPorSiniestro: (boolean - default: false)

    Indicador para recuperar o no la lista de información de siniestros por siniestro. Valores posibles: true (si), false (no). Valor por defecto: false.

    Indicador para recuperar la lista de información de siniestros por póliza o no. Valores posibles: true (sí), falso (no). Valor predeterminado: false. Si este parámetro se establece en true, se requiere idPoliza.

    Indicador para recuperar la lista de información de siniestros por parte involucrada o no. Valores posibles: true (sí), false (no). Valor predeterminado: false.

  • idPoliza: (string)

    Identificador único de la póliza. Concatenación del código de línea de negocio y el número de póliza. Si se especifica, recuperarPorPoliza debe debe ser true.
